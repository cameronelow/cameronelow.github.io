Notable Projects

Data Science

Coffee Quality Control Chart

The first version of this project was building a quality control chart for the Marine Depot Maintenance Command during my senior year of high school. I monitored cost abnormalities for marine equipment. Due to the confidentiality of this project, I updated it with data on prices for Arabica and Robusta coffee beans. In the demo, I created tabbed web application using R Shiny - the first tab ("Updated QCC") has an improved UI and automatically updates based on input where as the second tab ("Original QCC") closely mirrors my version 1 from high school.

Multi-Dimensional Analysis of Iris with PCA and t-SNE

For working with multi-dimensional user data, I used dimension-reduction techniques like PCA (principal component analysis) and t-SNE (t-distributed stochastic neighborhood embedding) to better visualize the data on a two-dimensional plane. The original project with CharacTour involved confidential datasets, so I recreated this analysis using the open-source Iris dataset in R.

DBSCAN with Iris

Another project I worked on for CharacTour was implementing DBSCAN (density-based clustering of applications with noise) to derive user insights. While the data is also confidential, I replicated it with the Iris dataset in R Through this unsupervised machine learning tactic, I was able to procure useful results to better understand complex user behavior.
